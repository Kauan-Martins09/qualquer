GITHUB: O GitHub é uma plataforma baseada na nuvem que funciona como a maior rede social e serviço de hospedagem de código-fonte do mundo.
GITLAB: O GitLab é uma plataforma completa de DevSecOps baseada na web que permite às equipes planejar, construir, proteger e implantar softwares em um único local.
BITBICKET: O Bitbucket é uma plataforma de hospedagem de código-fonte baseada em Git, desenvolvida pela Atlassian.
PULL REQUEST: Pull request (PR) é uma solicitação feita por desenvolvedores para mesclar (merge) alterações de código de uma ramificação (branch) para o repositório principal, geralmente no GitHub ou GitLab.
ISSUE TRACKI8NG: Issue Tracking (rastreamento de problemas) é um processo e sistema de software utilizado para registrar, priorizar, gerenciar e monitorar o status de tarefas, bugs, defeitos ou solicitações de melhorias em projetos.
